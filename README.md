# Another-weather-App
This is another weather App based on OpenWeatherMap data. Beautiful design, adaptive (of course) with cool icons (from weather icons) on board.
I also managed to design 2 custom functions. The first one detects wind strength based on Bofort scale, adopted by World Meteorological Organization.
The second function helps to calculate and set the reduction coefficient of the wind turbine depending on the speed of wind. This feature helps to sync
the rotation of wind turbine blades with wind speed.

# Ещё одно погодное приложение
Это еще одно погодное приложение, основанное на данных OpenWeatherMap. Красивый дизайн, адаптивный (разумеется) с прикольными иконками (из weather icons kit) на борту.
Мне также удалось создать 2 пользовательские функции. Первая функция определяет силу ветра на основе Шкалы Бо́форта, принятой Всемирной метеорологической организацией.
Вторая функция помогает рассчитать и установить коэффициент понижения скорости вращения лопастей турбины в зависимости от скорости ветра. Эта функция помогает синхронизировать
вращение лопастей ветряной турбины со скоростью ветра.
